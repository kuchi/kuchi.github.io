Systems Design for Graphic Presentation

This course is an introduction to the digital image and the hardware, software, and concepts necessary to understand how digital images are captured, stored, and processed. Basic image processing using MATLAB is also covered.

At the completion of this course you should understand what a digital image is and how digital systems handle images. You should be able to describe how basic image input and output systems work and understand basic image processing operations.
